## 1. 基础结构：用 `x` 画一条线

  示例代码会用两种方式可视化归一化后的 `x` 坐标（`st.x`）：
- **亮度方式**：直接把 `x` 当作灰度值，呈现从黑到白的渐变
- **绘图方式**：用一条绿色线条画出 `y = x` 的图像（即 `plot()` 函数)
这种 `x` 与 `y`（或亮度）的一对一对应关系，叫做**线性插值（linear interpolation）**。
```glsl
#ifdef GL_ES
precision mediump float;
#endif

uniform vec2 u_resolution;
uniform vec2 u_mouse;
uniform float u_time;

// 函数定义
float plot(vec2 st) {    
    //        ↑ 这里的 vec2 st 是"形参声明"
    return 1.0-smoothstep(0.0, 0.02, abs(st.y - st.x));
    // 原书写法：return smoothstep(0.02, 0.0, abs(st.y-st.x))
        // smoothstep(edge0,edge1,x),一般要求edge0<edge1,严格来说用例并不是规范写法
        // abs(edge0-edge1)= 线段宽度
    // 输出smoothstep的插值，范围在0.0~1.0内
}

void main() {
	vec2 st = gl_FragCoord.xy/u_resolution;
    float y = st.x;
    vec3 color = vec3(y);

    // 函数调用
    float pct = plot(st);
	color = mix(color, vec3(1.000,0.149,0.624), pct);
    // 原书写法：color = (1.0-pct)*color+pct*vec3(0.0,1.0,0.0)
        // mix函数用法见下一章节

	gl_FragColor = vec4(color,1.0);
}
```
**小提示**：

- `vec3(value)` 会自动把三个颜色通道都设为同一个值

- `vec4(vec3, value)` 用一个三维向量加一个值构造四维向量（第四个值通常控制 alpha）

## 2. 用数学函数为直线"塑形"

### 2.1 幂函数 `pow()`

把 `x` 进行 `pow(x, 5.0)` 之类的运算，直线就会变成曲线。

- 可以尝试不同的指数：`20.0`、`2.0`、`1.0`、`0.0`、`0.2`、`0.02`
- `pow()` 是 GLSL 原生函数，通常有硬件加速，合理使用能让代码更快

### 2.2 其他数学函数

可以替换成：

| 函数 | 作用 |
|------|------|
| [`exp()`](https://thebookofshaders.com/glossary/?search=exp) | 指数函数 |
| [`log()`](https://thebookofshaders.com/glossary/?search=log) | 对数函数 |
| [`sqrt()`](https://thebookofshaders.com/glossary/?search=sqrt) | 平方根 |

这些函数搭配 `PI`（代码中用宏定义为 `3.14159265359`）会有更有趣的效果。

## 3. Step 与 Smoothstep

GLSL 自带一些硬件加速的插值函数，非常实用。

### 3.1 `step(threshold, value)`

```glsl
#ifdef GL_ES
precision mediump float;
#endif

#define PI 3.14159265359

uniform vec2 u_resolution;
uniform float u_time;

float plot(vec2 st, float pct){
  return  smoothstep( pct-0.02, pct, st.y) -
          smoothstep( pct, pct+0.02, st.y);
}

void main() {
    vec2 st = gl_FragCoord.xy/u_resolution;

    // Step will return 0.0 unless the value is over 0.5,
    // in that case it will return 1.0
    float y = step(0.5,st.x);

    vec3 color = vec3(y);

    float pct = plot(st,y);
    color = (1.0-pct)*color+pct*vec3(0.0,1.0,0.0);
    //等同于color=mix(color,vec3(0.0,1.0,0.0),pct)

    gl_FragColor = vec4(color,1.0);
}
```
![[Pasted image 20260618190259.png]]
上面示例的代码包含渲染绿色线条的部分，单纯用到step函数的main部分只有：
```glsl
void main() {
    vec2 st = gl_FragCoord.xy/u_resolution;

    float y = step(0.5,st.x);
    
    vec3 color = vec3(y);
    gl_FragColor = vec4(color,1.0);
}
```
- 接收两个参数：**阈值（边界）** 和 **要检测的值**
- 小于阈值返回 `0.0`，大于等于阈值返回 `1.0`
- 效果是一条"硬边界"的阶跃曲线

### 3.2 `smoothstep(edge0, edge1, value)`

- 接收一个区间 `[edge0, edge1]` 和一个待插值的值
- 在区间内做平滑过渡，区间外分别钳制为 `0.0` 和 `1.0`
- 比 `step()` 更柔和，常用来制作平滑过渡

### 3.3 用两个 smoothstep 拼出一个"凸起"

```glsl
float y = smoothstep(0.2, 0.5, st.x) - smoothstep(0.5, 0.8, st.x);
```

把两个 `smoothstep()` 相减，可以在某个 `x` 位置画出一个"凸起（bump）"，这正是之前 `plot()` 函数画绿线用的技巧——把它想象成对图形做了一次"竖直切片"。

## 4. 正弦（Sine）与余弦（Cosine）

正弦和余弦是做动画、塑形、混合数值时最好用的工具

- 给定一个角度（弧度），`cos()` 返回单位圆上对应点的 *x* 坐标，`sin()` 返回 *y* 坐标
- 它们的输出始终在 **-1 到 1** 之间，且过渡非常平滑

### 实验练习

| 操作 | 效果 |
|------|------|
| 在算 `sin` 前给 `x` 加上 `u_time` | 波形沿 x 轴产生**运动（motion）** |
| 在算 `sin` 前给 `x` 乘以 `PI` | 周期**缩短**，每 2 个单位重复一次 |
| 用 `u_time` 乘以 `x` 再算 `sin` | **频率**越来越压缩（注意 `u_time` 可能已经很大，导致图像难以辨认） |
| 给 `sin(x)` 加 `1.0` | 整条波形向上**平移**，取值变为 0.0～2.0 |
| 给 `sin(x)` 乘以 `2.0` | **振幅**翻倍 |
| 对 `sin(x)` 取绝对值 `abs()` | 形成类似**小球弹跳**的轨迹 |
| 对 `sin(x)` 取小数部分 `fract()` | 产生锯齿状的周期波形 |
| 对 `sin(x)` 分别取 `ceil()` 和 `floor()` 并相加 | 得到只有 `1` 和 `-1` 两个值的"数字波" |

## 小结

| 概念 | 关键点 |
|------|--------|
| 线性插值 | `x` 与 `y`（或亮度）一一对应 |
| `pow()` / `exp()` / `log()` / `sqrt()` | 把直线变成各种曲线 |
| `step()` | 硬阈值，产生阶跃 |
| `smoothstep()` | 区间内平滑过渡 |
| 两个 `smoothstep` 相减 | 制造一个"凸起" |
| `sin()` / `cos()` | 周期性、平滑、值域 [-1, 1]，是动画与波形的核心工具 |
| `abs()` / `fract()` / `ceil()` / `floor()` | 进一步改造波形的形状 |
