```glsl
#ifdef GL_ES
precision mediump float;
#endif

uniform vec2 u_resolution;
uniform float u_time;

vec3 colorA = vec3(1.0, 0.6353, 0.8588);
vec3 colorB = vec3(1.0, 0.9529, 0.5137);

void main() {
    vec3 color = vec3(0.0);
    //初始化颜色color=(0.0,0.0,0.0)
    
    float pct= abs(sin(u_time));
    color=mix(colorA,colorB,pct);
    //mix(a, b, t) = a * (1 - t) + b * t，t=0时返回a，t=1时返回b，t在0和1之间时返回a和b的线性插值
    //pct=0时，color=colorA；pct=1时，color=colorB；

    gl_FragColor = vec4(color,1.0);
}
```

 uniform→定义颜色→**初始化**→渐变百分比→混合→输出
## mix函数

`mix (a, b, t )`
返回值 = `a * (1.0 - t) + b * t`

也就是在 `a` 和 `b` 之间根据 `t` 做线性插值：

- `t = 0.0` → 返回 `a`
- `t = 1.0` → 返回 `b`
- `t = 0.5` → 返回 `a` 和 `b` 的中间值
- `t` 也可以超出 `[0,1]` 范围，会做外插（extrapolation）

## 常见用法

### 1. 插值数字

```glsl
float result = mix(0.0, 10.0, 0.3); 
// 结果 = 3.0
```

### 2. 混合颜色

```glsl
vec3 color1 = vec3(1.0, 0.0, 0.0); // 红
vec3 color2 = vec3(0.0, 0.0, 1.0); // 蓝
vec3 color = mix(color1, color2, st.x); // 沿 x 轴从红渐变到蓝
```

`mix()` 对 `vec2`、`vec3`、`vec4` 都支持，会对每个分量分别插值。

### 3. `t` 用变量而不是常数，实现动态混合

```glsl
float pct = abs(sin(u_time));
vec3 color = mix(color1, color2, pct); 
// 在两色之间来回过渡
```

### 4. 搭配 `smoothstep()` 做更自然的过渡

```glsl
float pct = smoothstep(0.0, 1.0, st.x);
vec3 color = mix(color1, color2, pct);
```

比直接用 `st.x` 线性插值更柔和，因为 `smoothstep` 在两端会有缓入缓出的效果。

### 5. 多色混合（链式调用）

```glsl
vec3 colorA = mix(color1, color2, pct1);
vec3 color  = mix(colorA, color3, pct2);
```

通过多次 `mix()` 可以串联出多段渐变。

>[!summary] 总结
> `mix(a, b, t)` 就是"按比例 `t` 在 `a` 和 `b` 之间取值"，t 通常来自坐标、时间或其他塑形函数（如 `smoothstep`），是颜色渐变和动画过渡的核心工具。
